Testing the experimental Valhalla project of the JDK

## Try

 * Download a modified JDK: http://jdk.java.net/valhalla/
 * git clone https://github.com/karussell/test-valhalla-jdk
 * export JAVA_HOME=/path/to/jdk-14-valhalla/
 * mvn clean install
